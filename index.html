I made this site through many hours of googling. I started by first getting the basic layout of the grid using CSS grid and styling each of the areas' background colors. I then added the image of me and the text at the top using the img tag with a right float. After I added the picture of an otter, my favorite animal, to the center of area 1 using flex display and auto margin. Area 2 which contains my name and "phone number" used heading 1 and 2 tag and the phone icon was taken from Font Awesome. Section 4 is a table of some of my past classes which was constructed using the table tag + a bit of styling with :nth child and :hover to color the cells in different states. Area 5 has a png of a computer centered vertically and horizontally which uses similar horizontal aligning methods as area 2 but additionally uses transform to align vertically. Area 6 contains hyper links in a list using the "a" and "li" tags. Area 7 is dummy text which stylizes the text using b, code, and i tags to bold, code block, and italicize respectively. Lastly, the icons at the bottom contain links to their respective website using the "a" tag with the img tag.
